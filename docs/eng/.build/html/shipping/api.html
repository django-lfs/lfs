Shipping API¶

Models¶

class lfs.shipping.models.ShippingMethod(*args, **kwargs)¶

A shipping method can have several criteria which decide whether the shipping method is valid and hence offered for a customer to choose.

A shipping method can also have several prices. The first valid price for a shipping method is the actual price. If no assigned price is valid the default price is taken.

A shipping method can have an image and a note. These are displayed by default within the checkout page.

class lfs.shipping.models.ShippingMethodPrice(*args, **kwargs)¶

A shipping method price belongs to a shipping method and can have several criteria which decide whether the price is valid or not.
